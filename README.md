# my-portfolio-

This is Sarthak Hundare's personal portfolio website, built using React, Vite, Tailwind CSS, Framer Motion, GSAP, and Three.js.

## Tech Stack
- **Frontend Framework**: React (v19) & Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP, Framer Motion
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
